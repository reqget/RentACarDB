# RentACarDB

## Overview

This project is a web-based car rental system built with ASP.NET Core.
It consists of two parts: an API for backend operations and a UI for user interaction.

---

## Structure

* **RentACarDB** → Backend (API)
* **rentAcarView** → Frontend (UI)

---

## Technologies

* ASP.NET Core
* Entity Framework Core
* MySQL
* Bootstrap

## Usage

Run both projects to use the system:

```bash
cd RentACarDB
dotnet run
```

```bash
cd rentAcarView
dotnet run
```

---

## Description

The system allows listing cars and managing rental operations through a simple interface.
